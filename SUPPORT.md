# Support

Codex Clone Launcher is a community-maintained open-source desktop app.

## Where To Get Help

- Use GitHub Issues for reproducible bugs.
- Use the feature request template for product suggestions.
- Use the release/update template for installer, updater, signing, or
  `latest.json` problems.
- Check `README.md` and `docs/codex-clone-launcher.md` before opening a new
  issue.

## What To Include

Useful reports include:

- app version
- Windows version
- install or run method
- exact commands
- concise reproduction steps
- sanitized log excerpts
- whether the issue happens in a fresh clone profile

## Do Not Share Secrets

Do not post, attach, or paste:

- `auth.json`
- `.credentials.json`
- API keys, bearer tokens, OAuth codes, access tokens, or refresh tokens
- raw `sessions` or `archived_sessions`
- `state_5.sqlite` or other real profile databases
- full sync packages from a real profile
- screenshots with visible account secrets

Redact local paths when they reveal personal account names or private project
names.
